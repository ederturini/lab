# 🔬 Eder Turini Lab

Portfólio técnico público com projetos de IA aplicada, datasets comportamentais, experimentos e ferramentas.

🌐 **Site:** [lab.ederturini.com.br](https://lab.ederturini.com.br)

## Estrutura

```
lab/
├── index.html              # Página principal do laboratório
├── projects/
│   ├── senselens/          # Sistema de análise visual com IA
│   ├── oraculus/           # Motor preditivo com LLMs
│   ├── experiments/        # Laboratório de experimentação
│   └── ai-tools/           # Ferramentas e utilitários de IA

├── articles/               # Artigos e notas de pesquisa

└── demos/                  # Demonstrações interativas
```
