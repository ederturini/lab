# 📊 Datasets

Coleção de datasets estruturados para análise comportamental, predição de tendências e experimentos de IA.

## Datasets Disponíveis

### 🧠 Modelos Comportamentais
Datasets estruturados para análise de padrões comportamentais e predição de tendências em plataformas profissionais.

- **Formato:** CSV, JSON
- **Tamanho:** variável
- **Atualização:** mensal

### 📈 Dados de Engajamento
Métricas, scores e padrões de interação coletados e processados para análise de performance de conteúdo.

- **Formato:** JSON, Parquet
- **Tamanho:** variável
- **Atualização:** semanal

## Como Usar

```bash
# Clone o repositório
git clone https://github.com/ederturini/lab.git

# Acesse os datasets
cd lab/datasets/
```
